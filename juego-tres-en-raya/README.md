El juego es un juego de tablero 3*3, consite en la combinacion lineal de la misma figura o letra. La combinacion puede ser horizontal, vertical o diagonal.

El jugador que logre primero la combinacion gana, otra forma termina en empate.

Viendo el video sobre el juego, aprendi que para crear el juego se requiere de de la creacion de componentes estructurales en creados atraves del framework y logicos, en este caso se uso react para el framework y javascript para la parte logica.

La parte mas dificil fue seguir el video mientras imitaba la creacion del juego. Luego decidi reescribir el codigo en ves de solo copiarlo del github del tutorial, y tuve problemas para ver el juego por las configuraciones finales del codigo.
